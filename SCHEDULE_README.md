# 매일 오전 10시 자동 실행 설정 (Windows)

## 방법 1: 작업 스케줄러 등록 (권장)

**관리자 권한**으로 PowerShell 또는 CMD를 연 뒤 아래 명령을 실행하세요.

```powershell
schtasks /create /tn "Tiffany 재고 확인" /tr "C:\Users\User\Desktop\tiffany_stock\run_daily_check.bat" /sc daily /st 10:00 /ru "%USERNAME%"
```

- **오전 10시**에 매일 `run_daily_check.bat`이 실행됩니다.
- 배치 파일은 `monitor_size_5_8.py --once`를 실행해 **한 번만** 재고를 확인하고, 재고가 있으면 메일을 보낸 뒤 종료합니다.

### 작업 확인/삭제

- 확인: `schtasks /query /tn "Tiffany 재고 확인"`
- 삭제: `schtasks /delete /tn "Tiffany 재고 확인" /f`

## PC가 꺼져 있어도 실행되게 하려면?

### 1) 절전/슬립 상태라면 → 10시에 PC 깨우기 (가능)

PC를 **완전 종료**하지 않고 **절전(슬립)**으로 두면, 작업 스케줄러가 10시에 PC를 깨우고 작업을 실행할 수 있습니다.

**설정 방법 (작업 스케줄러 GUI):**

1. **Win + R** → `taskschd.msc` 입력 → 확인
2. 왼쪽에서 **작업 스케줄러 라이브러리** 선택
3. **"Tiffany 재고 확인"** 작업 더블클릭
4. **일반** 탭 → **"가장 높은 권한으로 실행"** 필요 시 체크
5. **조건** 탭 → **"이 작업을 실행하기 위해 컴퓨터 절전 해제"** 체크 → 확인

이렇게 하면 PC가 절전 상태여도 10시에 깨어나서 재고 확인이 실행됩니다. (전원 어댑터 연결 권장)

---

### 2) PC 전원을 완전히 끄는 경우 → 이 PC에서는 불가능

**전원이 완전히 꺼진 PC**에서는 어떤 프로그램도 실행할 수 없습니다. 대신 아래 중 하나가 필요합니다.

| 방법 | 설명 |
|------|------|
| **매일 10시에 전원 자동 켜기** | PC BIOS/설정에서 "예약 전원 켜기"(RTC Wake)를 오전 10시로 설정. 메인보드/노트북 지원 시 가능. |
| **항상 켜둔 기기에서 실행** | Raspberry Pi, 구형 노트북, 항상 켜두는 PC에 이 폴더를 복사한 뒤 그 기기에서 작업 스케줄러 등록. |
| **클라우드에서 실행** | GitHub Actions, AWS Lambda 등에서 매일 10시(한국 시간)에 스크립트 실행. Playwright 등 설정이 필요해 초기 작업이 다소 있음. |

원하시면 **GitHub Actions**로 매일 10시 KST에 돌리는 예시(워크플로 파일)도 만들어 드릴 수 있습니다. (무료, PC 전원과 무관)
